01 Visão Geral do Projeto

Presença online + mapa de distribuidores,
em uma solução integrada.

O projeto tem como objetivo criar uma presença institucional online para a empresa e facilitar que visitantes encontrem distribuidores por cidade, estado ou região — tudo em uma única página, clara e responsiva.

A solução será desenvolvida em WordPress, com um plugin personalizado para carregar os distribuidores no mapa, exibir informações e direcionar o usuário para WhatsApp ou rota no Google Maps.

02 Por que não usar a Google Maps API?

Leaflet + OpenStreetMap —
mais leve, sem custo recorrente.

A API completa do Google Maps pode gerar cobrança conforme o volume de acessos e recursos utilizados. Para esse tipo de projeto, a solução com Leaflet + OpenStreetMap entrega a mesma funcionalidade com menor risco financeiro e maior simplicidade técnica.

O botão de rota ainda pode abrir o Google Maps externamente — sem a necessidade de carregar toda a estrutura paga da API dentro do site. O melhor dos dois mundos, sem o custo.

Menor risco de custo recorrente
Sem billing mensal da API do Google Maps por volume de requisições ou acessos.
Mapa leve e responsivo
Leaflet é otimizado para performance e carrega rápido em qualquer dispositivo.
Sem dependência de API proprietária
OpenStreetMap é open-source. Sem chaves obrigatórias, sem quotas, sem surpresas.
Rotas via Google Maps externo
O botão "Ver Rota" abre o Google Maps externo — funciona sem nenhum custo de API.
Boa performance
Carregamento significativamente mais rápido que a API completa do Google Maps.
Ideal para o escopo atual
Entrega marcadores, busca, cards e botões de ação — exatamente o necessário.
